Settings

How far the per-room map embed (shown on a room's Location page) zooms in. {{ zoom_min }} = a tight crop to the room itself; higher values pull more of the surrounding floor into view. Range {{ zoom_min }}–{{ zoom_max }}; default {{ zoom_default }}. Only affects the cropped room embed, not whole-floor views.
How much page width the per-room embed takes up, as a percent of its column — the widget's footprint, independent of zoom (magnification is unchanged). Range {{ size_min }}–{{ size_max }}%; default {{ size_default }}%. Only affects the cropped room embed, not whole-floor views.
The shape of the per-room embed box. Vertical is a taller box; landscape is a short, wide box. Either way the crop is reshaped to fill the box with surrounding floor, keeping the room centred and fully visible. Default {{ orientation_default }}. Only affects the cropped room embed.
